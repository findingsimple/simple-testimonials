=== Simple Testimonials ===

Tags: testimonial, cpt, post, thumbnail
Requires at least: 4.0
Tested up to: 3.8.1
Stable tag: 1.0

Manage and display testimonials with your WordPress powered site.

== Description ==

Using WordPress Custom Post Types this plugin allows users to easily create, manage and 
display testimonials within a WordPress Powered site. 
 
= Features =

* **Manage** Create and manage testimonials using WordPress Custom Post Types
